∑ 計算方法と仕様の根拠

ダメージ計算

被ダメージ = 武器威力 × (100 − 防御補正)/100 × その他乗算カット

与ダメージ倍率 = (100 + 攻撃補正)/100

  • 攻撃補正（射撃 / 格闘）: 1pt = +1% の線形バフ。上限 100（= 与ダメ2倍）。
  • 防御補正（耐実弾 / 耐ビーム / 耐格闘）: 1pt = 1% の線形カット。上限 50（= 被ダメ50%減）。
  • 属性ダメージ軽減（オーバーチューン装甲系等）・部位カットは乗算で合成: 例 0.6 × 0.6 = 0.36。
  • 有効HP = HP ÷ (1 − 加重平均カット率)。
  • 三すくみは被ダメージ倍率に作用（有利＝被ダメ×0.8 → 有効HPは÷0.8で増加／不利＝被ダメ×1.3 → ÷1.3で減少）。与ダメージ側は有利×1.3／不利×0.8。

ステータス上限

HP 50000 / 耐実弾・耐ビーム・耐格闘 各50 / 射撃・格闘補正 各100 / スピード200 / 高速移動300 / スラスター100 / 旋回(地上・宇宙)各200。拡張スキルで上限が上昇する。

よろけ蓄積

射撃武器ごとに「よろけ値(%)」を持ち、累積が基準100%に達するとよろけが発生（被弾が3秒途切れると蓄積はリセット）。ダメージコントロール等のスキルは閾値を引き上げる（=より大きなよろけ値に耐える）。本ツールはスキルのよろけ値補正から「実質よろけ耐性」を算出する。

パーツ効果の解釈

  • 機体LV連動（オーバーチューン等）・強化段階連動（複合装甲材等）はLV・強化段階で効果量が変化する。
  • 機体HP +X%（特殊強化フレーム）は素の機体HPに対する割合で加算。補正 +X%（特殊強化装置）は補正値を乗算（×1.X）して上限でクランプ。
  • スキルのダメージカットは「常時」のもののみ既定でON。瀕死・静止・状態限定の効果は既定OFF（任意でON可）。

本ツールで計算に含めないもの

以下はゲーム機構上モデル化していない（数値は参考表示のみ／効果説明に記載）:

  • 部位HP（脚部/頭部/背部）の配分%・被弾時軽減%（部位HPの基礎値データが無いため集計不可）。
  • 能力UP「EXAM」等の任意発動・時限バフ、武器個別の威力・よろけ値・DPS。
